# IBM HR Attrition Intelligence Platform

A complete end-to-end data analytics project built on the **IBM HR Analytics Employee Attrition dataset** — covering the full pipeline from raw data to an interactive web dashboard, with SQL, Excel, and Power BI integration.

> **Dataset:** [IBM HR Analytics Employee Attrition & Performance](https://www.kaggle.com/datasets/pavansubhasht/ibm-hr-analytics-attrition-dataset) — Kaggle  
> **Rows:** 1,470 employees | **Columns:** 35 features | **Target:** Attrition (Yes/No)

---

## Live Preview

Run locally → **http://localhost:5000**

---

## Features

- **5-page interactive web app** — Home, Dashboard, Insights, Risk Predictor, Data Explorer
- **3 ML models** — Random Forest, Gradient Boosting, Logistic Regression with 5-fold CV
- **SHAP explainability** — feature-level prediction explanations
- **15 matplotlib/seaborn charts** — auto-generated by the pipeline
- **Risk scoring** — every employee assigned Low / Medium / High attrition risk
- **Scenario predictor** — enter any employee profile, get a risk score with gauge chart
- **SQL schema** — 3 stored procedures, 12 analytical queries, Power BI–ready view
- **Excel report** — 5 structured sheets (Employee Risk Data, Model Summary, Dept Breakdown, Role Breakdown, Feature Importance)

## Quick Start

### 1. Setup

```bash
git clone https://github.com/YOUR_USERNAME/ibm-hr-attrition-analytics.git
cd ibm-hr-attrition-analytics

python -m venv venv
venv\Scripts\activate          # Windows
# source venv/bin/activate     # Mac/Linux

pip install -r requirements.txt
```

### 2. Generate dataset & run ML pipeline

```bash
python scripts/create_ibm_dataset.py
python scripts/train_model.py
```

Takes ~2 minutes. Outputs 15 charts + Excel report + trained model.

### 3. Start web app

```bash
python web/app.py
```

Open → **http://localhost:5000**

### 4. MySQL (optional)

```bash
mysql -u root -p < sql/hr_attrition_schema.sql
python scripts/load_to_mysql.py --password your_password
```

### 5. Power BI (optional)

**Option A:** Get Data → Excel → `data/hr_attrition_report.xlsx` → load all sheets  
**Option B:** Get Data → MySQL → `ibm_hr_attrition` → `vw_attrition_dashboard`

## ML Pipeline Summary

| Step | Details |
|------|---------|
| Features | 30 features after dropping constants |
| Split | 80% train / 20% test (stratified) |
| Validation | 5-fold cross-validation |
| Class handling | `class_weight='balanced'` for imbalanced target |
| Explainability | SHAP TreeExplainer on 300-sample subset |

## Key Attrition Drivers (from SHAP & Feature Importance)

1. **OverTime** — single strongest predictor
2. **MonthlyIncome** — lower income = higher risk
3. **YearsAtCompany** — 0–2 years is the critical window
4. **JobSatisfaction** — satisfaction ≤ 2 elevates risk significantly
5. **DistanceFromHome** — longer commute = higher attrition
6. **Age** — younger employees leave more
7. **WorkLifeBalance** — WLB score of 1 is a strong signal
